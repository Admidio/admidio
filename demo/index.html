Fill Admidio demo data

This script fills your Admidio database with a set of sample data. It uses the database connection configured in the config.php file. The structure of the database always corresponds to an earlier Admidio version, so that an update via the Admidio Update Wizard is always necessary after filling.

Please make sure that you have a current backup of your database or you don't need the data in the database anymore. The script will delete all Admidio data that belongs to the table prefix configured in the config.php.
