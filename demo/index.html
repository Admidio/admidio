Fill Admidio demo data

This script will fill you Admidio database with a set of sample data. It use the database connection that is configured in the config.php.

Please make sure that you have a current backup of your database or you don't need the data in the database anymore.
The script will delete all Admidio data that belongs to the table prefix configured in the config.php.
